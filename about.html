개인정보, 안전하게 가리세요

MaskPrivacy는 이미지 속 개인정보를 브라우저에서 자동으로 감지·마스킹하는 완전 무료 도구입니다. 파일이 서버로 전송되지 않아 유출 걱정이 없습니다.

🖥️
완전 로컬 처리
모든 이미지 처리가 브라우저 안에서만 이루어집니다. 서버로 전송되는 데이터가 없습니다.
⚡
무료 사용
회원가입 없이 즉시 사용 가능합니다. 영구 무료로 제공됩니다.
🎯
자동 탐지
주민번호, 전화번호, 이메일, 계좌번호를 자동으로 감지하고 마스킹합니다.
✏️
수동 마스킹
자동 탐지 외에도 원하는 영역을 드래그하여 직접 마스킹할 수 있습니다.
🔄
개별 ON/OFF
탐지된 항목을 개별적으로 켜고 끌 수 있어 세밀한 조정이 가능합니다.
📥
즉시 다운로드
마스킹 완료된 이미지를 PNG 형식으로 즉시 저장할 수 있습니다.

어떻게 작동하나요?

1

이미지 업로드

마스킹이 필요한 이미지를 드래그하거나 클릭하여 업로드합니다. PNG, JPG, WEBP 형식을 지원합니다.

2

자동 개인정보 탐지

Tesseract.js OCR 엔진이 브라우저 내에서 이미지를 분석하고, 정규식 패턴으로 주민번호·전화번호·이메일·계좌번호를 탐지합니다.

3

마스킹 확인 및 조정

탐지된 항목을 목록에서 확인하고 개별 ON/OFF로 조정합니다. 누락된 영역은 드래그로 직접 추가할 수 있습니다.

4

다운로드

마스킹 완료된 이미지를 PNG로 즉시 다운로드합니다. 원본 이미지는 어디에도 저장되지 않습니다.
